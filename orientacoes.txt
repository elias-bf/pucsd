GIT - Software para controle de versão
Comandos:
	git init: defini a pasta atual como pasta versionada
	git status: mostra a situação dos arquivos na area de Stage e Work
	git ls-files: lista os arquivos monitorados pelo git
	git add: adiciona o arquivo para a area de Stage
	git commit: gera uma versão dos codigos que estao na area de Stage
	git log: mostrar o historico de commits
	git push: enviar os arquivos para o repositorio remoto
	git whatchanged: semelhante ao git log
	git clone: faz uma copia de um repositorio remoto para a pasta local
	git pull: atualiza o repositorio local com os dados do repositorio remoto
